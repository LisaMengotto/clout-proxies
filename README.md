# Clout Proxies

By Lisa Mangato

Live demo: [https://lisamengotto.github.io/clout-proxies/](https://lisamengotto.github.io/clout-proxies/)

Clout Proxies is a concept demo for a tool that helps people think about the
signals others use to judge credibility, reputation, taste, and professional
strength.

The current version is a lightweight intake and example-report experience. It
lets a user add links or PDFs, choose the categories they care about, and view
the kind of report the product could generate.

This repository contains both:

- the static demo used for GitHub Pages
- the early app scaffold for a fuller report-generation version
